# spark-count-fruit
A programming homework of cloud computing lesson.
## Spark 课后练习
### 具体任务:
1) 求出输入数据中最多人喜欢的水果</br>
2) 以及喜欢这种水果的人有哪些</br>
### 任务说明:</br>
1) 输入数据已给出,每行分别代表一个人以及他喜欢的水果</br>
2) 结果输出到 HDFS 上</br>
3) 提交的作业应包括:</br>
a) 可以成功编译运行的源码</br>
b) 关键运行步骤的截图</br>
c) 程序输出的文件(请从 HDFS 上下载下来)</br>
